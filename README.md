### Welcome to GridQuestRPG
A turn-based RPG with:
 an inventory system, 
 Mini-Games (such as Gambling)
 a Magic System 
 and a 2D world map!

 ### Instructions ###
 #### To Run Game, Run the GridQuestRPG.java file
 #### Commands include: go up, go down, go left, go right, attack, inventory, map, save, load, help, and quit (if unsure, enter 2 to view commands).
 #### Using the attack command starts a battle with a Goblin. The Goblin has HP, and you may use items during combat to help defeat it.
 #### After defeating the Goblin, you arrive at the Town Square, which contains multiple rooms that help you progress toward the final goal of reaching the castle.
 #### The Wizardâ€™s Arcane Study can be accessed from the Town Square. The Wizard offers magical trials that allow you to unlock special spells.
 - Trial of Flame requires answering riddles correctly to unlock the Fire Bolt spell.
 - Trial of Light requires donating potions to unlock the Healing Light spell.
 - Trial of Time requires defeating a summoned enemy to unlock the Time Slow spell.
 - Unlocked spells can be used during combat to gain an advantage against enemies.
 #### Town Square
 - From the Town Square, using the go left command takes you to the Gambling Room, where you must give up one potion to play. You will choose whether the next card is higher or lower, with a chance to win potions and continue with double-or-nothing until you lose.
 - From the Town Square, using the go right command takes you to the Abandoned Storage Room, which contains a chest.
 - From the Town Square, using the go up command takes you to the Forgotten Path.
 
 #### Forgotten Path
 - From the Forgotten Path, using the go down command returns you to the Town Square.
 - From the Forgotten Path, using the go up command takes you to the Castle.

 #### The Castle contains the final boss fight against the Sentinel, where you may use attacks and any items or spells you have unlocked.
 #### After defeating the Sentinel, the game is completed and you return to the Town Square.
